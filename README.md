# Klasifikasi Jenis Sampah Berbasis Citra Menggunakan Model deep learning ResNet-50 buatkan saya flowchart dari judul ini

# Deskripsi : 
- Hanya Bisa Memprediksi 4 Jenis Sampah Yaitu 
 - Biological
 - glass
 - plastic
 - paper
- Sistem ini mampu mengenali dan membedakan 4 jenis sampah, seperti Biological, glass, plastic, dan paper secara otomatis melalui analisis gambar. Dengan teknik preprocessing dan augmentasi data, model dilatih untuk mengekstraksi fitur penting tiap sampah sehingga menghasilkan klasifikasi yang akurat. Hasil penelitian ini dapat mendukung pengelolaan sampah yang lebih efisien dan ramah lingkungan.

# Fitur : 
- User mengunggah gambar jenis sampah diantara 4 jenis yang diatas 
- Tekan Button Prediksi Untuk menentukan jenis sampah nya
- Model melakukan Proses
- Output/Hasil nya Jenis Sampah dan Tingkat Kepastian

# Teknologi : 
- Backend : Menggunakan Python (Flask)
- Frontend : Menggunakan Javascript (React JS) dan CSS
- Deployment : Menggunakan Vercel

# Cara Instal Dan Gunakan Di Localhost : 
- git clone https://github.com/UasKelompokAstra/frontend
- New terminal Pada Text Editor
- cd UasKelompokAstra
- cd frontend
- ketik "npm install" untuk menginstal kembali library
- setelah itu ketik "npm run dev" Untuk menjalankan react js di localhost
- Maka Akan mendapatkan  "http://localhost:5174/"
- buka browser untuk melihat tampilan dari web nya
